utility/random.h

Wrapper for random number/string generators!

The following piece of code shows some examples of random number/string generator 

            // generates a random integer in range [0...N - 1] both inclusive
            int index = random_int_range(0, N - 1);

            // generates a value in range [0...1] both inclusive
            double probability = random_double();

            // generates a random token of length 24
            char* token = random_hex_string(24)

            // generates a 6-digit random sequence of [0...9]
            char* otp = random_decimal_string(6);

Function description
void randomize(void);
  • seeds the random number/string generator
  • since release @2020.1
int random_int(void);
  • generates a random integer
  • since release @2020.1
int random_int_upper_bound(int upper_bound);
  • generates a random integer in range [0...upper_bound] both inclusive
  • asserts that upper_bound >= 0
  • since release @2020.1
int random_int_range(int lower_bound, int upper_bound);
  • generates a random integer in range [lower_bound...upper_bound] both inclusive
  • asserts that 0 <= lower_bound and lower_bound <= upper_bound
  • since release @2020.1
double random_double(void);
  • generates a random double floating point value in range [0...1] both inclusive
  • since release @2020.1
double random_double_upper_bound(double upper_bound);
  • generates a random double-floating-point value in range [0...upper_bound] both inclusive
  • asserts that upper_bound >= 0
  • since release @2020.1
double random_double_range(double lower_bound, double upper_bound);
  • generates a random double in range [lower_bound...upper_bound] both inclusive
  • asserts that 0 <= lower_bound and lower_bound <= upper_bound
  • since release @2020.1
char* random_binary_string(int length);
  • generates a random binary string of given length on heap
  • asserts that length is a positive integer
  • since release @2020.1
