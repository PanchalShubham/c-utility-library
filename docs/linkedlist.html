utility/linkedlist.h

Doubly-linked-list implementation of list! Implements all optional list operations and permits all elements including NULL!
linkedlist is not syncronized!
All memory allocation operations are done on heap!
Time and space complexity of operations are described below!


The following example uses wrapper (from wrapper.h) for creating an linked list of integers! 

            // seed random number generator from random.h
            randomize();

            // create a new empty linked list of initial capacity 5
            LinkedList *list = create_empty_linked_list(5);

            // insert 7 elements in linked list in range [0...100]
            // note that size of the list will be expanded i.e. doubled when needed
            for (int i = 0; i < 7; ++i)
                linked_list_insert_back(list, wrap_int(random_int_upper_bound(100)));

            // prints the content of linked list to console using iterators
            printlinkedList(list);

            // delete the node with value 10 from list if exist
            // note the use of parameters free_key and free_list_key set to true
            bool deletionResult = linked_list_delete(list, wrap_int(10), compare_callback, true, true);
            // free_key set to true means the memory which is allocated on heap for wrapped integer will be freed at the end
            // and free_list_key set to true means that memory which was allocated on heap for wrapped integer while insertion should be freed
            /*free key options are supplied with primitive types and wrappers in mind*/
The following piece of code uses linked list iterator to traverse the linked list (note the use of wrapper.h again) 

            // prints the content of linked list to console
            void printlinkedList(LinkedList* list){

                // print size and capacity of list to console
                printf ("List(%d/%d): ", list->size, list->capacity);

                // create a new linked list iterator associated with given list
                LinkedListIterator* iterator = create_linked_list_iterator(list);

                // iterate as long as we have a next node
                while (linked_list_iterator_has_next(iterator))
                    printf ("%d ", wrapped_int_value(linked_list_iterator_next(iterator)));

                // free memory allocated to iterator
                free_linked_list_iterator(iterator);
            }
The following piece of code shows the example of callback (type included from utilitytypes.h) 

        /*returns -1, 0 or 1 according to if key1 is less than, equal or gerater than key2*/
        int compare_callback(void* key1, void* key2){

            // first we cast key1 and key2 pointers to their original types (int in this case)
            int val1 = *((int*)key1);
            int val2 = *((int*)key2);

            // next we perform a comparison and return the result back
            return (val1 > val2 ? 1  : val1 == val2 ? 0 : -1);
        }

Function description
LinkedList* create_empty_linked_list (int initial_capacity);
  • creates a new linked list on heap
  • returns pointer to newly created linkedlist if succeeds otherwise returns NULL
  • asserts that initial capacity is a positive integer
  • takes constant time
  • since release @2020.1
bool linked_list_insert_at (LinkedList* list, void* value, int index);
  • inserts value at given index in linked list
  • available indices are [0...N] where N is the size of the list
  • returns true if succeeds otherwise returns false
  • asserts that list, list's linked aren't NULL and index is in bound
  • takes linear time
  • since release @2020.1
bool linked_list_insert_front (LinkedList* list, void* value);
  • inserts value at front of linked list
  • returns true if succeeds otherwise returns false
  • asserts that list, list's linked aren't NULL
  • takes linear time
  • since release @2020.1
bool linked_list_insert_back (LinkedList* list, void* value);
  • inserts value at back of linked list
  • returns true if succeeds otherwise returns false
  • asserts that list, list's linked aren't NULL
  • takes amortized constant time
  • since release @2020.1
void* linked_list_delete_at (LinkedList* list, int index);
  • deletes value at given index in linked list
  • available indices are [0...N-1] where N is the size of the list
  • returns the value at given index
  • asserts that list, list's linked aren't NULL and index is in bound
  • takes linear time
  • since release @2020.1
void* linked_list_delete_front (LinkedList* list);
  • deletes value from front of linked list
  • returns the value at given index if exist otherwise returns NULL
  • asserts that list, list's linked aren't NULL
  • takes linear time
  • since release @2020.1
void* linked_list_delete_back (LinkedList* list);
  • deletes value from back of linked list
  • returns the value at given index if exist otherwise returns NULL
  • asserts that list, list's linked aren't NULL
  • takes amortized constant time
  • since release @2020.1
bool linked_list_delete(LinkedList* list, void* key, utility_compare compare_callback, bool free_key, bool free_list_key);
