utility/Stopwatch.h

Implementation of Stopwatch using system gettimeofday!
Designed for running time analysis of algoritms!
Stopwatch is not syncronized!
All operations are done in constant time!

The following example computes the time taken by quick-sort algorithm using Stopwatch! 

            // create a new Stopwatch instance
            Stopwatch* watch = create_stopwatch();

            // start Stopwatch before running algorithm
            stopwatch_start(watch);

            // run algorithm
            quick_sort(array, 0, N - 1);

            // stop Stopwatch after finishing algorithm
            stopwatch_stop(watch);

            // get the time taken by algorithm
            long long int time_taken = stopwatch_time_elapsed(watch);

            // free memory allocated to Stopwatch
            stopwatch_free(watch)

Function description
Stopwatch* create_stopwatch(void);
  • creates a new Stopwatch on heap
  • returns pointer to newly created Stopwatch if succeeds otherwise returns NULL
  • since release @2020.1
void stopwatch_start(Stopwatch* watch);
  • starts the Stopwatch i.e. start counting time
  • returns nothing!
  • asserts that watch is not NULL and isn't started before
  • since release @2020.1
void stopwatch_stop(Stopwatch* watch);
  • stops the Stopwatch i.e. stop counting time
  • returns nothing!
  • asserts that watch is not NULL and is running
  • since release @2020.1
long long int stopwatch_time_elapsed(Stopwatch* watch);
  • computes the time taken during its running period
  • returns time taken!
  • asserts that watch is not NULL and isn't running
  • since release @2020.1
void stopwatch_reset(Stopwatch* watch);
  • resets the Stopwatch
  • returns nothing!
  • asserts that watch is not NULL
  • since release @2020.1
void stopwatch_free(Stopwatch* watch);
  • frees the memory allocated to Stopwatch on heap
  • returns nothing!
  • asserts that watch is not NULL
  • since release @2020.1
