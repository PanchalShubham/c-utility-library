utility library

A small and simple library for C programming language!

Headers
arrays.h - Implements algorithms for arrays!
arraylist.h - Implements list operation using resizing-arrays!
linkedlist.h - Implements list operation using doubly linked lists!
queue.h - Implements queue operation using singly linked lists!
stack.h - Implements stack operation using singly linked lists!
random.h - Implements random number/string generators!
timer.h - Implements timer for performance analysis of algorithms!
types.h - Defines types for comparator and swappers!
wrapper.h - Provides wrappers for primitive type for ease of containers implementation!
